## 产品介绍

### 什么是PolyVault？

PolyVault是一款跨平台密码管理器，采用零知识加密架构，确保您的敏感信息安全存储。支持Windows、macOS、Linux、iOS和Android多平台同步。

### 核心特性

| 特性 | 说明 |
|------|------|
| **零知识加密** | 数据仅在您的设备上解密，服务器无法访问明文 |
| **硬件安全模块** | 支持TPM、Secure Enclave等硬件级安全存储 |
| **多端同步** | 支持多设备实时同步，无需第三方云服务 |
| **AI Agent集成** | 可选集成AI助手，智能填写表单 |
| **浏览器扩展** | 支持Chrome/Firefox/Edge自动填充 |
| **开源透明** | 核心代码开源，可审计安全性 |

### 系统要求

| 平台 | 最低要求 | 推荐配置 |
|------|---------|---------|
| **Windows** | Windows 10 | Windows 11 + TPM 2.0 |
| **macOS** | macOS 12 | macOS 14 + Apple Silicon |
| **Linux** | Ubuntu 20.04 | Ubuntu 22.04 |
| **iOS** | iOS 15 | iOS 17 + Face ID |
| **Android** | Android 10 | Android 14 + StrongBox |

## 快速开始

### 安装

#### Windows

1. 从官网下载安装包：`PolyVault-Setup.exe`
2. 双击运行安装程序
3. 选择安装目录，点击"安装"
4. 安装完成后启动PolyVault

#### macOS

1. 从App Store或官网下载 `PolyVault.dmg`
2. 双击打开磁盘映像
3. 将PolyVault拖入Applications文件夹
4. 首次运行需在"系统设置 > 隐私与安全性"中允许运行

#### Linux

```bash
# Debian/Ubuntu
sudo apt install ./polyvault_1.0.0_amd64.deb

# Fedora/RHEL
sudo dnf install ./polyvault_1.0.0_x86_64.rpm

# AppImage
chmod +x PolyVault-1.0.0-x86_64.AppImage
./PolyVault-1.0.0-x86_64.AppImage
```

#### 移动端

- **iOS**: App Store搜索"PolyVault"
- **Android**: Google Play商店或官网下载APK

### 6. 数据备份与恢复

#### 自动备份

PolyVault支持自动备份功能，保护您的数据安全：

```
设置 > 数据管理 > 自动备份
```

**备份选项**:

| 选项 | 说明 |
|------|------|
| **备份频率** | 每天 / 每周 / 每月 |
| **备份位置** | 本地 / iCloud / Google Drive / 自定义路径 |
| **保留版本** | 最近10个 / 30个 / 无限 |
| **加密备份** | 使用主密码加密备份文件 |

#### 手动备份

```
设置 > 数据管理 > 备份凭证
```

步骤：
1. 点击"备份凭证"
2. 选择备份位置
3. 设置备份密码（可选）
4. 等待备份完成

**备份文件格式**: `.pvbackup`（加密格式）

#### 恢复数据

```
设置 > 数据管理 > 恢复凭证
```

步骤：
1. 点击"恢复凭证"
2. 选择备份文件
3. 输入备份密码（如设置了加密）
4. 确认恢复

⚠️ **注意**: 恢复数据将覆盖当前设备上的所有凭证，建议先备份当前数据。

## 安全指南

### 权限级别

PolyVault采用三级权限系统：

| 级别 | 访问范围 | 验证方式 |
|------|---------|---------|
| **L1** | 基本凭证 | 主密码/生物识别 |
| **L2** | 敏感凭证 | 主密码 + 二次验证 |
| **L3** | 高敏感凭证 | 主密码 + K宝验证 |

### 安全建议

#### 主密码保护

- ✅ 使用至少12位混合字符
- ✅ 不要与其他网站共用密码
- ✅ 定期更换主密码（建议3-6个月）
- ❌ 不要写在纸上或保存在文件中
- ❌ 不要告诉任何人

#### 日常使用

- ✅ 离开电脑时锁定应用
- ✅ 启用自动锁定（建议5分钟）
- ✅ 定期检查登录历史
- ❌ 不要在公共电脑保存密码

#### 设备安全

- ✅ 启用设备加密
- ✅ 保持系统更新
- ✅ 安装可信的安全软件
- ❌ 不要越狱或Root设备

### eCAL自动授权规则

PolyVault的eCAL（Encrypted Cross-Device Access Layer）功能支持多设备间安全同步。您可以通过自动授权规则简化设备配对流程。

#### 访问自动授权设置

```
设置 > 安全设置 > 自动授权规则
```

#### 规则类型

| 规则 | 说明 | 推荐设置 |
|------|------|---------|
| **同网络自动授权** | 同一WiFi网络下自动授权信任设备 | 家庭环境开启，办公环境关闭 |
| **首次配对确认** | 新设备首次连接需用户确认 | 建议始终开启 |
| **授权有效期** | 授权凭证有效时间 | 24小时（默认） |
| **自动续期** | 到期前自动续期 | 开启 |

#### 使用场景配置

**场景1: 家庭环境**
```
配置:
- 同网络自动授权：✅ 开启
- 首次配对确认：✅ 开启
- 授权有效期：24 小时
- 自动续期：✅ 开启

效果: 家庭设备间自动授权，新设备需确认一次
```

**场景2: 办公环境**
```
配置:
- 同网络自动授权：❌ 关闭
- 首次配对确认：✅ 开启
- 授权有效期：8 小时
- 自动续期：❌ 关闭

效果: 每次连接都需确认，提高安全性
```

**场景3: 高安全环境**
```
配置:
- 同网络自动授权：❌ 关闭
- 首次配对确认：✅ 开启
- 授权有效期：1 小时
- 自动续期：❌ 关闭

效果: 每次授权仅1小时有效，最大化安全
```

#### 管理已授权设备

```
设置 > 设备管理
```

您可以：

### 紧急情况处理

#### 忘记主密码

**重要提示：** 主密码无法找回！

建议措施：
1. 如果开启了生物识别，使用生物识别解锁
2. 使用紧急访问功能（需提前设置）
3. 使用备份恢复（如有）

#### 设备丢失

1. 登录其他设备的PolyVault
2. 进入"设置 > 设备管理"
3. 撤销丢失设备的访问权限
4. 修改所有重要账户密码

#### 账户被盗

1. 立即修改主密码
2. 检查并撤销可疑设备
3. 检查登录历史
4. 如有必要，联系客服

## 常见问题

### Q1: 主密码忘记了怎么办？

A: 主密码无法找回。建议：
- 开启生物识别作为备用解锁方式
- 设置紧急访问联系人
- 定期导出数据备份

### Q2: 如何在多设备间同步？

A: PolyVault使用P2P加密同步：
1. 确保设备在同一局域网或网络可达
2. 在新设备登录相同账户
3. 首次同步需要验证主密码
4. 之后自动保持同步

### Q3: 数据存储在哪里？

A: 数据存储位置：
- 本地：加密存储在您的设备上
- 同步：P2P直接同步，不经过第三方服务器
- 云备份：可选开启，数据端到端加密

### Q4: PolyVault会看到我的密码吗？

A: 不会。PolyVault采用零知识架构：
- 所有加密在您的设备上进行
- 服务器只存储加密数据
- 只有您拥有解密密钥

### Q5: 如何导出数据？

A: 导出步骤：
1. 进入"设置 > 数据管理"
2. 选择"导出数据"
3. 选择格式（加密备份/JSON/CSV）
4. 输入主密码确认
5. 选择保存位置
