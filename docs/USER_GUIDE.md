# PolyVault 用户手册

**版本**: v1.0.0  
**更新时间**: 2026-03-24

---

## 简介

PolyVault 是一款跨设备凭证管理应用，帮助您安全地存储和管理密码、API密钥等敏感信息。

### 核心功能

- 🔐 **安全存储** - 军事级加密保护您的凭证
- 📱 **跨设备同步** - 在手机、平板、电脑间无缝同步
- 🔗 **设备管理** - 管理已登录设备，随时撤销访问
- 🛡️ **担保系统** - 去中心化信任验证

## 快速开始

### 1. 下载安装

| 平台 | 下载方式 |
|------|---------|
| iOS | App Store 搜索 "PolyVault" |
| Android | Google Play 或下载 APK |
| Windows | 官网下载安装包 |
| macOS | App Store 或官网下载 |
| Linux | Snap / AppImage / Deb |

### 2. 注册账户

1. 打开应用，点击"注册"
2. 输入邮箱地址
3. 设置强密码（至少8位，包含大小写字母和数字）
4. 选择用户类型（人类/AI）
5. 验证邮箱地址

### 3. 设置主密码

主密码用于加密您的所有数据：

- ⚠️ **请务必记住**：主密码无法找回
- 🔒 **建议**：使用密码短语（如：My-Dog-Loves-Treats-2024!）
- 💡 **提示**：记在纸上并存放在安全的地方

## 凭证管理

### 添加凭证

1. 点击右下角 **+** 按钮
2. 填写信息：
   - **服务名称**：如 GitHub、AWS
   - **用户名/邮箱**：登录账号
   - **密码**：点击生成强密码
   - **网址**（可选）：服务地址
   - **备注**（可选）：额外信息
3. 点击"保存"

### 密码生成器

点击密码字段旁的生成图标：

| 选项 | 说明 |
|------|------|
| 长度 | 8-64 字符 |
| 大写字母 | A-Z |
| 小写字母 | a-z |
| 数字 | 0-9 |
| 特殊字符 | !@#$%^&* |

### 搜索凭证

1. 在首页顶部搜索框输入关键词
2. 支持搜索：服务名、用户名、备注

### 编辑凭证

1. 点击凭证条目
2. 点击右上角"编辑"按钮
3. 修改信息后保存

### 删除凭证

1. 点击凭证条目
2. 点击右上角"删除"按钮
3. 确认删除

## 设备管理

### 查看已登录设备

1. 进入"设置" → "设备管理"
2. 查看所有已登录设备列表

设备信息包括：
- 设备名称和类型
- 最后活跃时间
- 是否为当前设备
- 是否为信任设备

### 移除设备

1. 在设备列表中找到要移除的设备
2. 点击"移除"按钮
3. 确认移除

### 信任设备

信任设备不需要频繁重新验证：

1. 点击设备旁的"信任"按钮
2. 输入主密码确认

## 数据同步

### 自动同步

凭证会自动在所有设备间同步：

1. 确保设备在线
2. 登录同一账户
3. 数据将自动同步

### 手动同步

1. 下拉刷新首页
2. 或点击"设置" → "立即同步"

### 同步冲突

当两个设备同时修改同一凭证时：

1. 应用会提示冲突
2. 选择保留哪个版本
3. 或手动合并更改

## 担保系统

### 什么是担保？

担保是 PolyVault 的信任验证机制。通过其他用户的担保，您可以：
- 建立信誉分数
- 解锁更多功能
- 参与社区治理

### 担保等级

| 等级 | 名称 | 条件 |
|------|------|------|
| 1 | 未担保 | 新注册用户 |
| 2 | 基础认证 | 1人担保 |
| 3 | 信任用户 | 3人担保 |
| 4 | 认证用户 | 5人高信誉担保 |

### 请求担保

1. 进入"个人主页" → "担保"
2. 点击"请求担保"
3. 选择担保人
4. 填写请求说明
5. 发送请求

### 为他人担保

1. 收到担保请求通知
2. 查看申请人信息
3. 点击"批准"或"拒绝"

⚠️ **注意**：为他人担保意味着您信任该用户，请谨慎操作。

## 安全设置

### 生物识别

启用指纹/面容解锁：

1. 进入"设置" → "安全"
2. 开启"生物识别解锁"
3. 验证身份

### 自动锁定

设置自动锁定时间：

| 选项 | 说明 |
|------|------|
| 立即 | 切换应用即锁定 |
| 1分钟 | 离开1分钟后锁定 |
| 5分钟 | 离开5分钟后锁定 |
| 从不 | 仅手动锁定 |

### 导出数据

1. 进入"设置" → "数据管理"
2. 点击"导出数据"
3. 输入主密码
4. 选择导出格式（JSON/CSV）
5. 安全保存导出文件

## 常见问题

### 忘记主密码怎么办？

**很遗憾，主密码无法找回。**

这是为了确保您的数据安全 - 只有您知道主密码，我们无法访问您的数据。

**建议**：
- 重新注册账户
- 从备份恢复数据（如果有）

### 数据存储在哪里？

- **本地**：加密存储在您的设备上
- **云端**：加密存储在我们的服务器
- **同步**：端到端加密，只有您可以解密

### 如何删除账户？

1. 进入"设置" → "账户"
2. 点击"删除账户"
3. 确认删除

⚠️ **警告**：删除账户将永久删除所有数据，无法恢复。
