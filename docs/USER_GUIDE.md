### 2. 基础配置

#### 2.1 安全设置

**路径**: 设置 → 安全

| 配置项 | 说明 | 推荐设置 |
|--------|------|---------|
| **生物识别** | 使用指纹/面容解锁 | ✅ 启用 |
| **自动锁定** | 离开后自动锁定 | ✅ 启用（5 分钟） |
| **锁定时长** | 自动锁定的等待时间 | 5 分钟 |
| **启动密码** | 启动时需要密码 | ✅ 启用 |
| **隐藏内容** | 后台时隐藏内容 | ✅ 启用 |

---

#### 2.2 通知设置

**路径**: 设置 → 通知

| 配置项 | 说明 | 推荐设置 |
|--------|------|---------|
| **授权请求** | 收到授权请求时通知 | ✅ 启用 |
| **设备发现** | 发现新设备时通知 | ✅ 启用 |
| **同步完成** | 数据同步完成通知 | ⚪ 可选 |
| **安全警报** | 异常访问警报 | ✅ 启用 |

---

#### 2.3 同步设置

**路径**: 设置 → 同步

| 配置项 | 说明 | 推荐设置 |
|--------|------|---------|
| **自动同步** | 自动同步到其他设备 | ✅ 启用 |
| **同步网络** | 同步时使用的网络 | Wi-Fi only |
| **同步频率** | 自动同步的时间间隔 | 实时 |

---

### 3. 权限级别使用场景

#### L1 普通级别

**适用场景**:
- ✅ 日常查看凭证列表
- ✅ 自动填充普通网站密码
- ✅ 接收消息通知

**限制**:
- ❌ 无法访问敏感凭证（银行、支付类）
- ❌ 无法修改配置
- ❌ 无法管理白名单

---

#### L2 增强级别

**适用场景**:
- ✅ 访问敏感凭证（需要生物认证）
- ✅ 添加普通 Agent 到白名单
- ✅ 接收远程控制请求
- ✅ 跨设备同步

**限制**:
- ❌ 无法访问关键凭证
- ❌ 无法修改自身权限
- ❌ 无法导出全部数据

---

#### L3 最高级别

**适用场景**:
- ✅ 访问所有凭证（包括关键凭证）
- ✅ 修改权限配置
- ✅ 管理完整白名单
- ✅ 导出/备份全部数据
- ✅ 设备重置

**限制**:
- ⏱️ 仅 5 分钟有效期
- 🔐 需要 K 宝 + 生物双重认证

---

## K 宝使用指南

### 1. 什么是 K 宝？

**K 宝**是一种硬件级安全验证设备（类似银行 U 盾），用于最高级别的安全认证。

```
┌─────────────────────────────┐
│  K 宝类型                    │
│                             │
│  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │ USB │  │ NFC │  │蓝牙 │ │
│  │ K 宝 │  │ K 宝 │  │ K 宝│ │
│  └─────┘  └─────┘  └─────┘ │
│                             │
│  特点：                     │
│  • 硬件级安全               │
│  • 动态令牌生成             │
│  • 防篡改设计               │
│  • 一次性使用               │
│                             │
└─────────────────────────────┘
```

## 常见问题

### Q1: 忘记主密码怎么办？

**A**: 主密码无法恢复，但可以重置：

```
1. 在登录界面点击"忘记密码"
2. 回答安全提示问题（如设置）
3. 通过邮箱验证
4. 重置主密码

⚠️ 注意：重置后会清除所有凭证数据！
```

---

### Q2: 生物识别失败怎么办？

**A**: 尝试以下方法：

```
1. 确保手指干燥清洁
2. 重新录入指纹/面容
3. 使用备用解锁方式（密码）
4. 重启应用

如仍失败，请联系技术支持。
```

---

### Q3: K 宝无法连接怎么办？

**A**: 检查以下项目：

```
USB K 宝:
• 检查 USB 接口是否松动
• 尝试其他 USB 接口
• 检查设备管理器中是否识别

NFC K 宝:
• 确保手机 NFC 已开启
• 移除手机壳
• 调整 K 宝位置

蓝牙 K 宝:
• 确保蓝牙已开启
• 检查电量是否充足
• 重新配对设备
```

---

### Q4: 多设备同步失败怎么办？

**A**: 检查网络连接：

```
1. 确保所有设备在同一网络
2. 检查防火墙设置
3. 重启 eCAL 服务
4. 手动触发同步

如仍失败，检查日志文件。
```

---

### Q5: Agent 白名单已满怎么办？

**A**: 清理不用的 Agent：

```
1. 进入"设置" → "Agent 白名单"
2. 移除不再使用的 Agent
3. 添加新 Agent

白名单上限：50 个 Agent
```

---

### Q6: 权限级别自动降级太快怎么办？

**A**: 调整降级时间：

```
路径：设置 → 安全 → 权限管理

L2 有效期：[30 分钟 ▼]
L3 有效期：[5 分钟 ▼]

可调整范围：
L2: 10-60 分钟
L3: 1-30 分钟
```
