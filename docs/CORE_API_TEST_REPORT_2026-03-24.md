## 6. 数据模型

### 6.1 凭证类型 (CredentialType)
- PASSWORD - 密码凭证
- OAUTH - OAuth凭证
- API_KEY - API密钥
- CERTIFICATE - 证书凭证

### 6.2 设备状态 (DeviceStatus)
- ONLINE - 在线
- OFFLINE - 离线
- PENDING - 待确认
- REVOKED - 已撤销

### 6.3 同步状态 (SyncStatus)
- SYNCING - 同步中
- COMPLETED - 已完成
- FAILED - 失败
- PENDING - 待处理
